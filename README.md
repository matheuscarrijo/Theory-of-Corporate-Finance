# Theory of Corporate Finance — Lecture Notes

This repository contains the lecture notes for the course **Theory of Corporate Finance**.

The repository includes:

- the compiled PDF version of the notes;
- the LaTeX source files;
- bibliography files;
- figures and other supporting material.

## Reading the lecture notes

If you simply want to read the notes, open the PDF file:

**Theory_of_Corporate_Finance.pdf**

You do not need to compile the LaTeX files yourself.

## Contributing to the lecture notes

Students are invited to contribute by suggesting corrections, comments, complements, and improvements.

There are two main ways to contribute.

## 1. Open an Issue

Use an **Issue** if you want to:

- report a typo;
- ask a question about an unclear passage;
- suggest a missing reference;
- propose an additional explanation;
- point out a possible mathematical or economic mistake;
- make a general comment about the notes.

When opening an Issue, please indicate as precisely as possible:

- the chapter or section;
- the page number, if relevant;
- the equation, proposition, definition, or paragraph you are referring to;
- your comment or suggestion.
